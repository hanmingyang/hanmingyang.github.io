hexo

什么hexo

hexo是一个基于Node.js的静态博客程序。可以快速的生成静态页面并能托管在github上。

为什么选择hexo

看下作者来自台湾的Tommy351的原话：

  • 不可思议的快速 ─ 只要一眨眼静态文件即生成完成
  • 支持 Markdown
  • 仅需一道指令即可部署到 GitHub Pages 和 Heroku
  • 已移植 Octopress 插件
  • 高扩展性、自订性
  • 兼容于 Windows, Mac & Linux

怎样搭建hexo博客

准备工作（一）：安装Git

安装之前可以确认一下本地是否安装了git，打开命令行输入git


- 安装[homebrew](http://brew.sh/index_zh-cn.html)，然后通过homebrew再来安装Git.
- 通过mac系统的Xcode安装

准备工作（二）：安装Node.js

准备工作（三）：安装hexo

> - 使用命令行：npm install -g hexo 
> - 创建hexo文件夹
> - 进入文件夹
> - 使用命令行：hexo init
> - 根据提示安装依赖包：npm install
> - 生成静态页面：hexo generate 或者 hexo g
> - 启动hexo服务：hexo server

好了，至此，本地博客已经搭建起来了.只是本地的
